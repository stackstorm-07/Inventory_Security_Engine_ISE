# Inventory Security Engine ISE
A secure web-based Inventory Management System (IMS) integrated with core cybersecurity principles to protect data, users, and operations from real-world threats.

Objectives of developing software/application:
1. This project focuses on the design and implementation of a secure web-based Inventory Management System (IMS) integrated with a comprehensive cybersecurity framework.
2. The proposed system will implement core cybersecurity principles such as authentication, Authorization, Confidentiality, Integrity, and Accountability.
3. The framework will include secure login mechanisms, Role-Based Access Control (RBAC), password security enforcement, database protection techniques, logging and monitoring of user activities, and backup mechanisms.
4. The project aims not only to develop a functional inventory system but also to demonstrate how security controls can be systematically integrated into web-based applications to mitigate real-world cyber threats.

Key Features:
1. Secure Login and Authentication Module
2. Role-Based Access Control (RBAC) Module
3. User and Role Management Module
4. Inventory and Supplier Management Module
5. Secure Password Storage (Hashing & Validation)

User Roles:
1. Admin: Full system control, manage users, roles, and permissions.
2. Manager: Update stock details, manage suppliers, and generate reports
3. Staff / Employee: View inventory data, and Limited access (no sensitive modifications)
